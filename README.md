# DATDEV INSTALLER

Tool cài phần mềm tự động.

## Chức năng bản v1
- Cài Google Chrome
- Cài WinRAR
- Cài UniKey
- Cài Brave
- Cài Microsoft Office 365
- Cài nhiều phần mềm cùng lúc
---
## Cách sử dụng


1. Tải file `install.exe`
2. Chuột phải
3. Chọn `Run as administrator`
4. Chọn phần mềm muốn cài
